WebGL Content Test

This page tests Remarq's behavior with WebGL-rendered 3D content. Like Canvas, WebGL renders directly to a pixel buffer with no DOM text structure.

Standard DOM Text (Annotatable)

Select and annotate this text normally. The spinning cube below is rendered via WebGL, a low-level graphics API that draws triangles, not text nodes.

WebGL 3D Scene (Not Annotatable)

Expected behavior: The canvas element above displays a rotating 3D cube. Attempting to select the canvas surface should not trigger annotation. Text around it works normally.

Technical Note

WebGL, like Canvas 2D, operates on pixel buffers. Even if text is rendered onto a WebGL surface (e.g., via texture mapping), it's not selectable DOM content. Annotation systems designed for 3D/spatial content (e.g., AR/VR annotation tools) use 3D coordinates or mesh surface identifiers, which is a fundamentally different approach.
